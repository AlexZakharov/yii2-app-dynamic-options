# Yii2 Application Dynamic Options
## Installation

Installation with Composer

Either run
~~~
    php composer.phar require kfosoft/yii2-app-dynamic-options:"*"
~~~
or add in composer.json
~~~
    "require": {
        ...
        "kfosoft/yii2-app-dynamic-options":"*"
    }
~~~
